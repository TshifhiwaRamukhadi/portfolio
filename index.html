TutorFind Platform

A comprehensive full-stack tutoring platform that connects students with qualified tutors for their registered modules. This was my final-year IT Project 3, achieving 87% (Distinction).

Key Features:

  • Multi-User System: Separate dashboards for Students, Tutors, and Admins with role-based access control
  • Tutor Management: Tutor registration, application system, module specialization, and availability scheduling
  • Session Booking: Real-time session requests, approval workflow, and session history tracking
  • Payment Integration: Secure wallet system, payment processing, and earnings management for tutors
  • Communication: Real-time chat messaging between students and tutors using SignalR
  • Review System: Student ratings and reviews for tutors with analytics
  • Notifications: Push notifications for session requests, approvals, and messages
  • Admin Dashboard: System statistics, user management, announcements, and analytics

Technical Architecture:

  • Backend: ASP.NET Core Web API (RESTful) with Entity Framework Core
  • Database: Microsoft SQL Server with comprehensive data models
  • Mobile App: Native Android application built with Java in Android Studio
  • Authentication: JWT-based authentication with secure password hashing
  • Real-time Features: SignalR hubs for chat and notifications
  • API Integration: Seamless communication between Android app and backend API
  • Repository Pattern: Clean architecture with service layer and data repositories
  • Middleware: Error handling, request logging, and JWT validation

Technologies Used:

ASP.NET Core 8.0, C#, Entity Framework Core, SQL Server, Android Studio, Java, SignalR, JWT, RESTful API, Git, Visual Studio
